Iubeo

Iubeo is a Project Management Framework based on Eclipse which allows you to:

Manage Projects, Requirements and Resources directly from your favourite development environment

Plan as you code! Define Gantt Charts and WBS through textual languages with syntax highlighting and content assist

Generate Documents: Generate nicely formatted Project Charters, Project Management Plans and Project Status Reports always update with your models.

WBS and Requirements

A set of Domain Specific Languages, Editors and Views to model and graphically depict Work Breakdown Structures And Project requirements. Define project requirements in separated files; then import them in WBS models and assign set of requirements to each Work Package. Import WBS files in schedules and assign work packages to planned activities.

View details

Resources

A Domain Specific Language to model classes and single resources, their role and costs. Resources model can be used along schedule models to assign single activities to different resources and calculate the whole resources exploitation and costs.

View details

Schedules

A Domain Specific Language called Plan allowing you to define plans and schedules as easily as you code. See graphically depicted as a Gantt the plan meanwhile you define it. Assign activities to resources and map activities to work package of the WBS. Automatically generate project charter, project management plan and project status reports.
